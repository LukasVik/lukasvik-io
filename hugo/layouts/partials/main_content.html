Open-source projects

tsfpga

I am the author and maintainer of tsfpga, a flexible and scalable development platform for modern FPGA projects. See the website for more information: https://tsfpga.com

hdl-modules

I am the author and maintainer of hdl-modules, a collection of reusable, high-quality, peer-reviewed VHDL building blocks. See the website for more information: https://hdl-modules.com/

hdl-registers

I am the author and maintainer of hdl-registers, an HDL register code generator fast enough to run in real time. See the website for more information: https://hdl-registers.com/
